PRE-REQUISITES:
===============

- Windows: Local admin privileges (for creating symlinks)
- Linux/MacOS: Extraction tool with .rar support (unrar, unar, Unarchiver etc.)
- X-Plane 10 or later
  

RECOMMENDED ADDON SCENERY (OPTIONAL):
=====================================

- Simheaven's X-Europe 3 or later
- Orthophoto mesh ZL16 or better


INSTALLATION:
=============

Windows:

- Run self-extracting .exe and follow instructions

Linux/MacOS:

- Extract .rar archive into X-Plane's "Custom Scenery" folder
