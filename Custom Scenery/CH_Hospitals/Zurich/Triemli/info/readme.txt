REQUIREMENTS:
=============
- OpenSceneryX 2.0 (or later)

Optionally, but highly recommended:

- X-Europe: https://simheaven.com/x-europe


INSTALLATION:
=============
1. Unzip the directory "Custom Scenery" into your X-Plane home.
2. Adjust scenery load order as needed. See:
   http://wiki.x-plane.com/Prioritization 
   http://developer.x-plane.com/2012/09/scenery-packs-the-new-rules/

Optionally for Ortho4XP mesh patching:

3. Copy the content of folder "Ortho4XP" to your Ortho4XP home folder, overwriting existing files.
4. Create orthophoto tile for +47+008 using preferred settings.
